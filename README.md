# Сайт «Фокус-группа ЭзоДвиж»

Продающий лендинг для группы по продвижению эзотериков (таро, астрология, нумерология, ведьмовство).
Одностраничник на чистом HTML/CSS — открывается двойным кликом по `index.html`, без сборки и зависимостей.

## Что внутри

| Файл | Назначение |
|------|------------|
| `index.html` | Весь сайт: разметка + стили + микро-скрипт карусели |
| `img/` | Все изображения (hero, иконки, кейсы, фоны) |
| `Лендинг-текст.md` | Исходный текст лендинга по экранам |

## Структура лендинга (9 экранов)

1. **Hero** — вырезанное фото с неоновым кольцом на тёмном космосе, заголовок «ЭЗО ДВИЖ» частично за моделью
2. **Боли** — узнавание себя
3. **Кто я + Кейсы** — 5 кейсов учениц в виде **таро-карт** разных оттенков с «редкостью»
4. **Что это** — суть группы
5. **Программа** — 8 модулей в горизонтальной **карусели**, у каждого 3D-иконка
6. **Результат** — точка Б
7. **До / После**
8. **Как попасть** — шаги + цена + заявка на созвон
9. **Частые вопросы** — с 3D-значками вопроса
+ финал с кнопкой **«Написать Софи»** (Telegram)

## Дизайн

- **Стиль:** тёмный «космос», фиолет + неон-розовый акцент (в тон кольцу на hero)
- **Шрифты:** Unbounded (заголовки), Manrope (текст), Space Mono (лейблы), Playfair Display (имена на таро-картах)
- **Mobile-first:** вёрстка под телефон, липкая кнопка заявки снизу на мобильных
- **Фоны секций:** затемнённые эзотерические (таро, луна, свечи, магический круг) и денежные (купюры, монеты, иконки Instagram) изображения

## Ассеты (img/)

- `hero-cut.png` — модель с кольцом, вырезана на прозрачный фон (хромакей)
- `hero.jpg` — исходное фото с розовым кольцом
- `case-*.jpg` — скрины кейсов учениц (Ира, Юля, Катя, Вика, Зыкова)
- `icon-*.png` — 8 3D-иконок модулей (фиолетовый рендер) + `icon-question.png`
- `bg-eso.png`, `bg-money.png` — тёмные фоны секций
- `star.png` — запасной кадр

Изображения сгенерированы в **Krea** (gpt-image-2 / kling), дизайн — с навыком ui-ux-pro-max.
