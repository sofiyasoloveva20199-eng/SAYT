# Сайт «Фокус-группа ЭзоДвиж»

Лендинг для курса по продвижению эзотериков. Одностраничник на чистом HTML/CSS.

- `index.html` — сам сайт (тёмный «космос», карусель модулей, кейсы)
- `img/` — hero (вырез на прозрачном фоне), скрины кейсов учениц
- `Лендинг-текст.md` — исходный текст по экранам

Открыть: двойной клик по `index.html`.
